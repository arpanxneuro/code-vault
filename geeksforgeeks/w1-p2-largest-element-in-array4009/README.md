## 📝 Problem Statement

Given an array `arr[]`. The task is to find the largest element and return it.

---

---

## 📥 Input

- An array `arr[]` of size `n`.

**Constraints:**

- `1 <= arr.size() <= 10^6`
- `0 <= arr[i] <= 10^6`

---

---

## 📤 Output

- Return the largest element from the array.

---

---

## ✅ Examples

Input:
5
1 2 3 4 5
90
Output:
1 2 3 4 5 90
